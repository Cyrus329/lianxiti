# 专升本题库 v39

本版本在 v38 校正与错题规划版基础上补回了可见的“导入本地备份”功能。

## 备份与恢复

- 电脑端：页面顶部可直接点击“导入备份”或“导出备份”。
- 手机端：进入“我的”→“题库与备份”，可选择“导出本地备份”或“导入本地备份”。
- 统计页的备份卡片也同时提供导出和导入按钮。
- 导入支持 v34—v39 题库导出的 JSON 备份文件。
- 导入前会提示确认；确认后会恢复做题记录、错题本、错因、学习统计和设置。
- v39 新导出的备份还会保存手动题源设置和自动生成的类似题。

## 使用方法

解压后直接打开 `index.html`，或把文件夹中的全部文件上传到 GitHub Pages。不要只上传 `index.html`。
